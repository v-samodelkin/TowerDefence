Программа для поиска всех пар в слове.
Пример запуска:
pairs -f text.txt -w nowords.txt -g -c 25 -a
Выведет 25 самых частых идущих подряд слов с учётом порядка и без учёта границ предложений.

Справку по всем ключам можно получить командой 
pairs --help
или 
pairs -h

Запуск без аргументов запустит чтение из sys.stdin
